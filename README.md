# таблица с выбором цвета ячеек

### Проект на gh-pages:

https://elensmith.github.io/vue-color-table/

### Функционал:

- замена цвета ячейки в таблице с использованием Vue Swatches,
- выбрать цвет, выбрать номер ячейки
